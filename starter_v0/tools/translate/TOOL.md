---
name: translate
description: Translate text between languages. Default: English → Vietnamese. Uses MyMemory free API.
api: MyMemory Translation API (no key required, 5000 chars/day free)
args:
  text: text to translate (max 500 chars per call)
  target_lang: target language code (default "vi" for Vietnamese)
  source_lang: source language code (default "en" for English)
---

No API key required. Free tier: 5000 chars/day.
Common lang codes: en, vi, zh, ja, ko, fr, de, es.
